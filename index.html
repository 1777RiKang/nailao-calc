📚 奶块计算器使用说明

🎯 基本用法

  • 在「基础属性」页面配置装备、附魔、精灵、Buff 等参数，右侧属性面板和伤害数字实时更新
  • 切换到「伤害分析」标签查看各职业技能详细伤害，包括龙系、弓系、影袭/暗影、法杖/灼烧/冰系、触须/镰刀/魔灵、猎魔人、占星师、戒指技能等
  • 选择世界 → 副本 → BOSS 后，系统根据武器 T 级与怪物防御等级自动匹配穿透率，伤害表显示的数值已做穿透折算
  • 所有设置（下拉框、预设、主题偏好）自动保存到浏览器本地存储，关闭页面后再次打开自动恢复
  • 预设栏支持 3 个独立配置槽位 + 当前状态，可保存、读取、一键清除，支持导出为 .txt 文件备份或跨设备导入
  • 预设改名 — 双击预设名称（如"预设1"）可自定义命名，按 Enter 保存，按 Esc 取消

⚙️ 核心计算

  • 全职业伤害模拟 — 覆盖龙骑士、射手、影刃、魔导师、召唤师、猎魔人、占星师七大职业，每个职业技能独立建模
  • 穿透系统 — 基于武器 T 级（T1-T8）× 怪物防御等级查表计算穿透率，不同副本的破甲比例自动匹配
  • 战斗效果 — Buff 区可开关破甲（物理×1.3）、反馈（远程×1.3）、易燃（全部×2.0，庇护/生命花环除外），三个效果独立判定可叠加
  • 平A宝石 — 刺击（猎魔刺剑）、瞄准（猎魔火铳）、晦暗（占星普攻），初级/中级/高级三档，直接写入基础伤害公式
  • Buff / 药水 / 宝石 — 共 17 项可配置参数，包括屏障/烟雾、穿刺/扩散、契约符石、传说技能、琳的祝福、过热、统御、阵营、稳固等，完整覆盖游戏内增伤体系
  • 精灵与戒指 — 精灵选择及等级、羁绊、防御/攻击技能、金苹果；戒指区支持物理/远程/魔法 buff、魔抗/护甲 buff、戒指特效等
